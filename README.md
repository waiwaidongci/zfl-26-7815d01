# 夜间灯塔守望

直接打开 `index.html` 即可游玩。方向键旋转光束，空格切换光束宽度，护送所有船只抵达右侧港湾。

## 游戏玩法

- **← / →** 旋转光束
- **空格** 切换光束宽度（宽/窄）
- **P** 暂停游戏
- **R** 重开本关

## 关卡系统

游戏内置 3 个关卡，难度递进。通关后自动解锁下一关。

### 自定义关卡

在主菜单点击「关卡编辑器」可创建和编辑自定义关卡：
- 放置暗礁和雾区
- 调整船只数量和起始位置
- 设置港湾位置
- 配置天气事件（海风、浓雾、灯塔闪烁）

### 关卡包导入导出

支持将自定义关卡打包导出为文件，或导入他人分享的关卡包。

#### 导出关卡包

1. 进入「选择关卡」界面
2. 点击「📤 导出关卡包」按钮
3. 填写关卡包名称、作者、说明
4. 勾选要导出的关卡（支持全选/全不选）
5. 点击「导出文件」，浏览器会自动下载 `.zfl26pack.json` 文件

也可以在编辑器中点击「📤 导出当前关卡」快速导出单个关卡。

#### 导入关卡包

1. 进入「选择关卡」界面
2. 点击「📥 导入关卡包」按钮
3. 选择导入方式：
   - **从文本粘贴**：粘贴关卡包 JSON 数据
   - **选择文件**：选择本地 `.zfl26pack.json` 文件
4. 解析成功后可查看关卡包信息
5. 选择重复关卡处理策略：
   - **跳过重复**：保留本地已有的同 ID 或同名关卡（默认）
   - **覆盖重复**：用导入的关卡数据和最佳分覆盖本地
   - **重命名导入**：为重复关卡生成新 ID，保留两者
6. 点击「确认导入」完成

#### 关卡包格式

关卡包为标准 JSON 格式，包含以下字段：

```json
{
  "formatVersion": 1,
  "gameVersion": "1.0.0",
  "packName": "我的关卡包",
  "author": "作者名",
  "description": "关卡包说明",
  "exportedAt": 1234567890123,
  "levels": [
    {
      "id": "custom_xxx",
      "name": "关卡名称",
      "boats": 3,
      "boatStartY": 130,
      "boatSpread": 330,
      "rocks": [],
      "fog": [],
      "harbor": { "x": 900, "top": 70, "bottom": 530 },
      "weather": { ... }
    }
  ],
  "levelBests": {
    "custom_xxx": 1000
  }
}
```

**版本兼容说明**：
- `formatVersion` 为关卡包格式版本号
- 导入时会校验版本，版本过新需要更新游戏
- 低版本关卡包可在高版本游戏中正常导入

## 其他功能

- **成就系统**：追踪游戏成就和统计数据
- **航海日志**：记录每次航行的详细信息
- **回放系统**：支持查看对局回放
- **游戏设置**：音效、亮度、灵敏度等可调
