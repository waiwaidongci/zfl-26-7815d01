# 夜间灯塔守望

直接打开 `index.html` 即可游玩。方向键旋转光束，空格切换光束宽度，护送所有船只抵达右侧港湾。

## 游戏玩法

- **← / →** 旋转光束
- **空格** 切换光束宽度（宽/窄）
- **P** 暂停游戏
- **R** 重开本关

## 关卡系统

游戏内置 3 个关卡，难度递进。通关后自动解锁下一关。

### 自定义关卡

在主菜单点击「关卡编辑器」可创建和编辑自定义关卡：
- 放置暗礁和雾区
- 调整船只数量和起始位置
- 设置港湾位置
- 配置天气事件（海风、浓雾、灯塔闪烁）
  - 海风：船只漂移加剧，可调整持续时间、冷却时间、漂移强度和速度加成
  - 浓雾：光束射程缩短，可调整持续时间、冷却时间、射程缩减和雾气浓度
  - 灯塔闪烁：光束角度抖动，可调整持续时间、冷却时间、抖动幅度和闪烁强度
  - 开启天气后点击「试玩关卡」可立即体验当前配置的效果
- 保存后的天气配置在以下场景中正确生效：
  - **选择关卡**：关卡卡片显示天气图标，进入关卡时加载天气
  - **航线战役**：战役进度会根据难度倍率调整天气持续时间与冷却时间
  - **自定义航线**：章节卡片聚合显示所含关卡的实际天气类型，而非根据难度硬编码
  - **关卡包导入导出**：天气配置随关卡数据一起导出和导入，保留完整参数
  - **回放记录**：回放中录制 `weatherStart`/`weatherEnd` 事件，播放时实时渲染雾气覆盖、风线、光束射程缩减和闪烁效果

### 关卡包导入导出

支持将自定义关卡打包导出为文件，或导入他人分享的关卡包。

#### 导出关卡包

1. 进入「选择关卡」界面
2. 点击「📤 导出关卡包」按钮
3. 填写关卡包名称、作者、说明
4. 勾选要导出的关卡（支持全选/全不选）
5. 点击「导出文件」，浏览器会自动下载 `.zfl26pack.json` 文件

也可以在编辑器中点击「📤 导出当前关卡」快速导出单个关卡。

#### 导入关卡包

1. 进入「选择关卡」界面
2. 点击「📥 导入关卡包」按钮
3. 选择导入方式：
   - **从文本粘贴**：粘贴关卡包 JSON 数据后点击「解析数据」
   - **选择文件**：选择本地 `.zfl26pack.json` 文件
4. 解析成功后可查看关卡包信息与关卡预览列表，每个关卡显示：
   - 🚢 船只数量、🪨 暗礁数量、🌫️ 雾区数量
   - 天气摘要（🌬️海风 / 🌫️浓雾 / ⚡闪烁 / 🌤️晴朗）
   - 最佳分
   - ⚠️ 重复状态标记（与本地关卡同 ID 或同名时显示）
5. 勾选要导入的关卡（支持「全选」/「全不选」，默认全选）
6. 选择重复关卡处理策略：
   - **跳过重复**：保留本地已有的同 ID 或同名关卡（默认）
   - **覆盖重复**：用导入的关卡数据和最佳分覆盖本地
   - **重命名导入**：为重复关卡生成新 ID，保留两者
7. 点击「确认导入」完成
8. 导入完成后，关卡选择、自定义航线可选关卡及最佳分显示会立即刷新

#### 关卡包格式

关卡包为标准 JSON 格式，包含以下字段：

```json
{
  "formatVersion": 1,
  "gameVersion": "1.0.0",
  "packName": "我的关卡包",
  "author": "作者名",
  "description": "关卡包说明",
  "exportedAt": 1234567890123,
  "levels": [
    {
      "id": "custom_xxx",
      "name": "关卡名称",
      "boats": 3,
      "boatStartY": 130,
      "boatSpread": 330,
      "rocks": [],
      "fog": [],
      "harbor": { "x": 900, "top": 70, "bottom": 530 },
      "weather": {
        "seaBreeze": {
          "enabled": true,
          "minDuration": 8,
          "maxDuration": 18,
          "minCooldown": 15,
          "maxCooldown": 30,
          "driftMultiplier": 2.0,
          "speedMultiplier": 1.15
        },
        "denseFog": {
          "enabled": false,
          "minDuration": 10,
          "maxDuration": 22,
          "minCooldown": 20,
          "maxCooldown": 40,
          "beamRangeMultiplier": 0.55,
          "fogOpacity": 0.5
        },
        "lighthouseFlash": {
          "enabled": false,
          "minDuration": 5,
          "maxDuration": 12,
          "minCooldown": 18,
          "maxCooldown": 35,
          "jitterAmount": 0.08,
          "flickerIntensity": 0.4
        }
      }
    }
  ],
  "levelBests": {
    "custom_xxx": 1000
  }
}
```

**版本兼容说明**：
- `formatVersion` 为关卡包格式版本号
- 导入时会校验版本，版本过新需要更新游戏
- 低版本关卡包可在高版本游戏中正常导入

## 其他功能

- **成就系统**：追踪游戏成就和统计数据
- **战绩分析**：深入分析每一次航行的表现
  - 支持按关卡名称搜索
  - 支持按失败原因筛选（偏离航道、光束迷航、撞上暗礁等）
  - 支持按得分和耗时排序
  - 详情展示天气事件时间线、船只损失原因
  - 标识是否来自战役模式，是否观看过回放
  - 提供回放入口和重新挑战功能
  - 统计摘要包含胜率、平均得分、平均用时、失败原因分布
- **回放系统**：支持查看对局回放，回放数据持久化存储
- **游戏设置**：音效、亮度、灵敏度等可调
